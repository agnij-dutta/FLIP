# Flare Network Configuration

## RPC / Chain IDs
- Flare Mainnet: `https://flare-api.flare.network/ext/C/rpc` (chainId 14)
- Flare Mainnet WSS: `wss://flare-api.flare.network/ext/C/ws`
- Coston2 Testnet: `https://coston2-api.flare.network/ext/C/rpc` (chainId 114)
- Coston2 WSS: `wss://coston2-api.flare.network/ext/C/ws`

## Explorers
- Mainnet: https://flarescan.com, https://flare-explorer.flare.network
- Coston2: https://coston2-explorer.flare.network

## Faucet
- Coston2 faucet: https://faucet.flare.network/coston2

```

FLARE_COSTON2_RPC=https://coston2-api.flare.network/ext/C/rpc
FLARE_MAINNET_WSS=wss://flare-api.flare.network/ext/C/ws
FLARE_COSTON2_WSS=wss://coston2-api.flare.network/ext/C/ws
FLARE_COSTON2_FAUCET=https://faucet.flare.network/coston2
```
